Para visualizar a aplicação, utilize o terminal.

Para utilizar a aplicação, você precisará de:
- Qualquer versão Python acima de 3.0, o qual pode ser obtido através do link: https://www.python.org/downloads/
- Um editor de código-fonte para executa-lo em seu terminal.

Para executar a aplicação, entra na pasta do projeto e insira o comando no seu terminal: python crud.py run
